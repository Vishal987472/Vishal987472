## About Me

I'm a full stack software engineer with an MCA from Chandigarh University, focused on building production-grade, scalable systems across the **Java / Spring Boot** and **MERN / Next.js** ecosystems. My work spans backend architecture, API design, database modeling, and modern frontend engineering — most recently a complete **Hospital Management System** with role-based access control across Admin, Doctor, Patient, and Receptionist workflows.

I care about writing clean, maintainable, well-tested code and designing systems that hold up under real-world constraints — not just demos. Alongside core backend and frontend work, I integrate **AI/LLM capabilities** (Gemini API) into product features, treating AI as an engineering tool rather than a gimmick.
